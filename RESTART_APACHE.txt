========================================
   CARA RESTART APACHE - XAMPP
========================================

LANGKAH 1: Buka XAMPP Control Panel
   - Cari icon XAMPP di desktop atau Start Menu
   - Double-click untuk membuka

LANGKAH 2: Stop Apache
   - Cari baris "Apache" 
   - Klik tombol "Stop" (warna merah)
   - Tunggu sampai status berubah

LANGKAH 3: Tunggu 3-5 detik

LANGKAH 4: Start Apache
   - Klik tombol "Start" (warna hijau)
   - Tunggu sampai Apache running

LANGKAH 5: Verifikasi
   - Buka browser
   - Akses: http://localhost/galery_sachi/public/check-php-limits.php
   - Harus muncul tanda ✅ hijau

LANGKAH 6: Test Upload
   - Login ke admin
   - Buka: Galeri → Tambah Galeri
   - Upload foto 5-10MB
   - Klik Simpan

========================================

Jika masih error, coba:
1. Restart komputer
2. Buka XAMPP Control Panel lagi
3. Start Apache
4. Test upload lagi
